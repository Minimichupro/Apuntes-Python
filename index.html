PRINT()

El print, sirve para "escribir" lo que quieras. Puedes usarlo para cualquier cosa que sea que aparezca como texto o números.
Siempre debo escribir entre comillas, si no, no lo reconocerá y dará error.
Si no quiero escribir texto, puedo escribir solo números pero ten en cuenta:
Si pones comillas, lo reconocerá como texto.
Si es sin comillas, como un número,
pongamos un ejemplo:

while True:

print("Hola, esto es lo que se va a mostrar al usuario")

print("Esto es texto 00112843")

print("Lo siguiente es un número")

print(010291)

En este script, hay 3 tipos distintos de print(), uno con texto solo, otro con texto y números que se reconocen como texto y otro con solo números.

DEF (FUNCIONES)

El "def" al que yo le llamo "definir", sirve para "crear" o "definir" funciones, es decir, crear un conjunto de acciones recopiladas en un mismo lugar para luego, simplemente introducir su nombre y no tener que pegar todo el rato el codigo que queremos que se realice
pongamos un ejemplo:

def saludo():
print(Hola, que tal estás?)

def si():
print(¡Claro!)

def no():
print(No, ¡lo siento!)

while True:

print(¿Qué quieres que haga?) print(Decir "hola", "afirmar" o "rechazar")
respuesta = input("> ").strip().lower()

if respuesta == "" or respuesta == respuesta.isdigit():
print("No te he entendido, ¡volvamos a empezar!")
elif respuesta == "hola":

elif respuesta == "afirmar":
si

En este caso, primero preguntamos que quiere, luego, cogemos su respuesta y dependiendo de esta, llamará a la función definida anteriormente para ejecutar el codigo.

INPUT()

El input, sirve para "cojer" lo que introduce el usuario. Esto sirve para coger sus respuestas y compararlas con algo o simplemente para que no se ejecute todo de una y deba apretar el "Enter" para que continue el código,
pongamos un ejemplo:

while True:

print("Texto que tu quieras")

respuesta = input("(s/n)").strip().lower()

if respuesta == "" or respuesta == respuesta.isdigit():
print("No válido")
elif respuesta == "n":
print("Oh, vale")
elif respuesta == "s":
print("¡Fantástico!")

En este script, hemos hecho que tome la respuesta del usuario y lo compare para ver que responder.
